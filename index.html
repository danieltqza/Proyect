HTML

HTML (Lenguaje de marcado de hipertexto o HyperText Markup Language por sus siglas en inglés) se basa en etiquetas. Es muy fácil de aprender lo básico, tiene muchas características como semántica, entender el contenido, identación, fácil de usar. No se considera un lenguaje de programación, a diferencia de CSS, pero sí a JavaScript.
HTML, que significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar utilizado para crear y estructurar el contenido de las páginas web. HTML proporciona una forma de describir la estructura de un documento web utilizando etiquetas y elementos predefinidos.

CSS

Hecho para decorar las páginas de HTML, ya que no tenían estilos, pero con la llegada de CSS se pudo dar más estilo gráfico a las páginas web.
CSS, que significa Cascading Style Sheets (Hojas de Estilo en Cascada), es un lenguaje de estilo utilizado para definir el aspecto y la presentación de documentos HTML (o XML). CSS describe cómo se deben mostrar los elementos HTML en la pantalla, en papel o en otros medios.

JavaScript

Después de HTML y CSS, las páginas, yo diría, eran estáticas pero no del todo. Sin embargo, con la llegada de JavaScript, se logró dar mayor interactividad al usuario (DOM) con la página Document Object Model.
JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos que se utiliza principalmente en el desarrollo web para crear interactividad y dinamismo en las páginas. Es un componente esencial de la programación del lado del cliente, lo que significa que se ejecuta en el navegador web del usuario final.

React

React es una biblioteca de JavaScript de código abierto que se utiliza principalmente para construir interfaces de usuario (UI) interactivas y basadas en componentes. Fue desarrollado por Facebook y lanzado en 2013. A lo largo de los años, se ha convertido en una de las herramientas más populares y ampliamente utilizadas para el desarrollo de aplicaciones web modernas.

¿Por qué hice esta bitácora?

Fue hecha para avanzar en conocimientos de diseño web, de manera autodidacta. En YouTube encontré muchos canales que disponían de cursos gratis de HTML, CSS, JavaScript, React, Astro, SQL y muchos más.

Por el momento, me centraré en HTML, CSS y un poco en JavaScript, para tener los fundamentos básicos de diseño web, y luego seguir aprendiendo más, como por ejemplo React, Astro, JavaScript (más avanzado), SQL, Node, etc.

¿Qué me gustaría lograr?

Me gustaría aprender JavaScript, React, SQL y Node.js en lo que tiene relación con diseño web. Aparte, en programación, me gustaría aprender Java, Python, JavaScript también, y SQL también.

Desafíos y obstáculos

Aunque recién estoy empezando en el ámbito del diseño web, mi mayor desafío soy yo. Debido a que no dispongo de una disciplina muy buena, me desconcentro muy rápido.

HTML

HTML (Lenguaje de marcado de hipertexto o HyperText Markup Language por sus siglas en inglés) Se Basa En Etiquetas Es Muy Facil De Aprender Lo Básico, Tiene Muchas Características Como Semantica, Entender El Contenido, Identación, Facil De Usar, No Se Considera Un Lenguaje De Programacion Igual Que A CSS Pero A JavaScript Si
HTML, que significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar utilizado para crear y estructurar el contenido de las páginas web. HTML proporciona una forma de describir la estructura de un documento web utilizando etiquetas y elementos predefinidos.

CSS

Hecho Para Decorar Las Páginas De HTML, Ya Que No Tenian Estilos Pero Con La LLegada De CSS Se Pudo Dar Mas Estilo Gráfico A Las Páginas Web
CSS, que significa Cascading Style Sheets (Hojas de Estilo en Cascada), es un lenguaje de estilo utilizado para definir el aspecto y la presentación de documentos HTML (o XML). CSS describe cómo se deben mostrar los elementos HTML en la pantalla, en papel o en otros medios.

JavaScript

Despues de HTML y CSS, Las Páginas Yo Diría Que Eran Estaticas Pero No Del Tedo, Pero Con la Llegada de JavaScript, Se Logro Dar Mayor Interactividad Al Usuario (DOM) Con la Pagina Document Objet Model
JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos que se utiliza principalmente en el desarrollo web para crear interactividad y dinamismo en las páginas. Es un componente esencial de la programación del lado del cliente, lo que significa que se ejecuta en el navegador web del usuario final.

React
React es una biblioteca de JavaScript de código abierto que se utiliza principalmente para construir interfaces de usuario (UI) interactivas y basadas en componentes. Fue desarrollado por Facebook y lanzado en 2013. A lo largo de los años, se ha convertido en una de las herramientas más populares y ampliamente utilizadas para el desarrollo de aplicaciones web modernas.
